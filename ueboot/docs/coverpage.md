# ueboot

> 概要介绍

- 前后端分离、Maven多模块、分层架构。
- 适用后台管理系统、移动应用接口等各种系统
- 提供常用组件（前后端都有），快速上手
- 提供脚手架代码，快速搭建环境并进行二次开发
- 提供前端UI组件，后台管理系统的CRUD分分钟搞定
- 提供代码生成器，不做重复劳动
- 完善的帮组文档和丰富使用示例，不愁看不懂
